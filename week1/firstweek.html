Course Overview

Full Stack Development encompasses both client-side (front-end) and server-side (back-end) development. This course will cover:

  • HTML5 & CSS3
  • JavaScript (ES6+)
  • Node.js & Express
  • Database Systems
  • React Framework
  • RESTful APIs
  • DevOps Basics
  • Security Fundamentals

Technology Stack Comparison

Layer Technologies Purpose
Front-End HTML, CSS, JavaScript User Interface & Client-side Logic
Back-End Node.js, Python, Java Server-side Logic & Processing
Database MySQL, MongoDB, PostgreSQL Data Storage & Management

Client-Server Model

The foundation of web applications where client devices request services from servers over a network.
